# en-leigame-resource

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面资源。这些页面各自独立运行，不依赖特定域名或外部网站，仅作为前端页面存档与展示用途。

## 目录说明

```
en-leigame-resource/
├── pages/          # 存放独立的 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── docs/           # 相关文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，可直接在浏览器中打开使用。
- `assets/`：存放页面所需的公共或私有资源，按需引用。
- `docs/`：可放置额外的开发说明、使用指南等文档。

## 页面归档说明

- 所有页面均为静态 HTML，无后端依赖。
- 每个页面独立完整，互不干扰。
- 页面归档遵循原始结构，不修改其内容与功能。
- 如页面需要引用外部资源（如字体、图标库），建议尽量使用本地版本或稳定 CDN。

## 维护说明

- 本仓库仅用于归档与发布，不提供在线动态服务。
- 欢迎提交 Issue 或 Pull Request 来修正页面错误、更新资源或新增归档。
- 请勿在本仓库中存放与页面无关的隐私数据、密钥或敏感信息。
- 仓库维护者会定期检查页面可用性与资源完整性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `pages/` 目录下的 HTML 文件即可浏览对应页面。
3. 如需部署到静态服务器，将整个仓库内容上传即可。
